# lightPad

A native iOS and iPadOS app that turns your device into a professional film viewing lightbox. It provides an adjustable, color-accurate backlight for scanning or viewing photographic film negatives and slides.

## Features

- **Adjustable Color Temperature**: Fine-tune the backlight from 3000K to 7000K, or use quick presets for B&W (5000K), Daylight/C-41 (5500K), or Tungsten (3200K).
- **Brightness Control**: Override system brightness directly within the app, including a quick "Max" toggle and an Auto-Max feature that immediately sets the screen to full brightness when the app is active.
- **Keep Awake**: Prevent the device screen from locking or dimming while viewing or scanning film.
- **Smart Interface**: The UI automatically adapts to the selected color temperature for optimum contrast and readability, and controls auto-hide to get out of your way.

## Building

### Prerequisites

- iOS / iPadOS 16.0+
- Xcode 15+
- [xcodegen](https://github.com/yonaskolb/XcodeGen) (optional but recommended)

### With xcodegen

```bash
brew install xcodegen
cd lightpad
xcodegen generate
open Lightpad.xcodeproj
# Cmd+R to build and run
```

### Without xcodegen (building from scratch)

The `.xcodeproj` is gitignored since it's generated by xcodegen. If you don't want to install xcodegen, you can recreate the project manually:

1. Open Xcode → File → New → Project → iOS → App
2. Replace the generated files with `LightpadApp.swift` and `ContentView.swift`
3. Add `Assets.xcassets`, `LaunchScreen.storyboard`, and `Info.plist` to the project bundle.
4. Point the target's Info.plist File setting to `Lightpad/Info.plist`.
5. Build and run.

## Recommendations for Best Color Accuracy

To ensure the most stable and accurate color temperature when using the lightbox:
1. Disable **True Tone** and **Night Shift** in iOS Control Center.
2. In the iOS Settings app under Accessibility → Display & Text Size, ensure **Auto-Brightness**, **Reduce White Point**, and **Color Filters** are turned off.
